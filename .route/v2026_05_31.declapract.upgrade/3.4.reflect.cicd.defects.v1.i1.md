## classification

### defect 1: biome format error in isOfDomainObject.test.ts

**classification:** repo quirk

**rationale:**

this defect was caused by code added in stone 3.1 (test refactor) that was not formatted via biome before commit. the root cause is workflow, not infrastructure:

- biome formatter is configured correctly
- format check CI is configured correctly
- developer did not run `npm run fix:format` before commit

this is not a practice bug because:
- the practice (biome format check) works as intended
- the practice caught the error and prevented merge
- no change to declapract-typescript-ehmpathy is needed

this is not an adoption candidate because:
- format enforcement is already in place
- pre-commit hooks could enforce this locally, but are optional

**lesson:** always run `npm run fix` before commit to catch format issues locally.
